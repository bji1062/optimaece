FAQ

Q1. 보안은 어떻게 보장하나요?

+

요청시 NDA(비밀유지협약서) 체결 가능합니다.

Q2. 어떤 데이터베이스 지원하나요?

+

SQL Server, MySQL, MariaDB, Aurora MySQL, Oracle을 지원합니다.

Q3. 복잡한 쿼리도 가능한가요?

+

1차 분석 후 난이도와 소요 시간을 안내드립니다.

Q4. 쿼리가 아닌 테이블 구조 문제도 해결하나요?

+

1차 분석 후 테이블 구조 변경이 필요한 경우 안내드리며, 원하시는 경우 개선 방향을 제시해드립니다.

Q5. 환불 정책은 어떻게 되나요?

+

30% 이상 성능 향상이 없으면 100% 환불해 드립니다.

Q6. 급한데 24시간도 기다릴 수 없어요!

+

긴급옵션(8시간 내 해결)도 제공 가능합니다. 50% 할증 적용됩니다.

Q7. 적용 후 문제가 생기면요?

+

튜닝 후 일주일간 무료 A/S 제공합니다. 적용 관련 문의사항도 언제든 질문해주세요.

Q8. 회사 승인이 필요한데 견적서 발행 가능한가요?

+

네, 가능합니다.
